Detailed Description

Mapeo de pines y definiciones de hardware.

Author
Tomas Canistra & Nogueira Ezequiel
  • Contiene las definiciones de macros para interactuar con los pines del Inversor V4.0 (Buzzer, Fan, Entradas ADC, PWM). También define la frecuencia del cristal para las rutinas de delay.
